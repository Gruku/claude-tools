Playwright tests mutate `.fixture-kanban/` via real API calls (auto/control, viewer.json prefs, sessions). Always revert after a test run:

```
git -C <worktree> checkout -- .fixture-kanban/
rm <worktree>/.fixture-kanban/.taskmaster/auto/sessions/*.events.jsonl
```
