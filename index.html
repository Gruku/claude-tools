Claude Tools

Plugin marketplace & tools for Claude Code by Gruku
Install marketplace: /plugin marketplace add Gruku/claude-tools
Then install any plugin: /plugin install plugin-name@gruku-tools

Plugins

Taskmaster v1.0.0
Universal AI-powered task and backlog management. Drop into any project — tracks tasks, epics, milestones, sessions, and dependencies. Includes kanban viewer, quality gates, worktree isolation, and TODO auditing.
new task management backlog mcp server
/init-taskmaster /start-session /pick-task /review-gate /end-session /check-todos
Shader Nodes v2.0.0
UE5.5 shader development pipeline. HLSL design (Opus) + Material Graph generation (Sonnet). Outputs UE5 clipboard format — paste directly into the Material Editor.
ue5 hlsl material graph
/shader-nodes:create /shader-nodes:yaml /shader-nodes:registry
Reality Reprojection v1.3.0
Web design system plugin. Generate components, convert existing code, review compliance — all following the Reality Reprojection design token system.
design system css web
/reality-reprojection:apply /reality-reprojection:generate /reality-reprojection:convert
Image Gen v1.0.0
Multi-backend image generation. Generate, edit, and iteratively refine images using Google Gemini and OpenAI GPT Image 1.5. Supports transparent PNGs, multi-turn refinement, and game asset creation.
gemini openai image generation
/image-gen /image-gen:generate /image-gen:edit /image-gen:refine
Codex Dispatch v1.0.0
Dispatch tasks to OpenAI Codex CLI for parallel execution with GPT/o-series models. Diversity of thought, cost-optimized sub-tasks, and independent parallel work.
multi-model codex wsl
/codex-dispatch
