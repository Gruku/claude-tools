# Volo's Feedback & Preferences — Reality Reprojection

All direct quotes and paraphrased preferences from the design iteration sessions.

---

## Grid & Grain

- "The grid is too invasive. I don't actually feel it." — Remove the grid overlay entirely.
- The invasiveness is "due to its overlay on top of everything" — it sat on top of all content.
- "Use the grain on some elements more sparingly." — Grain should never be global. Apply to specific elements only.
- "Lets introduce more material treatments with grains, and some concrete texture backgrounds." — Wants variety in grain density, not just one texture.
- "Add that grain to certain places barely visible around edges, but felt." — Grain should be subliminal, not visible. You feel it, you don't see it.
- "Especially we should add grains to gradients." — Every gradient should get grain to break CSS banding.

## Typography & Fonts

- "All the fonts except Syne are too narrow." — Syne is the winner for Declaration voice.
- "Barlow Condensed looks interesting and so is Archivo Black but Syne is just better." — Acknowledged alternatives but Syne remains the choice.
- "Lets experiment with Declaration font with a dropdown as well." — Wants the font selector dropdown kept for ongoing experimentation.
- "All the fonts are looking too small" — Wanted the page scaled to 140% of current size.

## Shadows & Light Mode

- "I actually like more how the shadows look in light mode than its dark mode alternative." — Light mode shadows are preferred. Make them richer, more layered.

## Favorite Elements

- "My favourite element is D1 — Value showcase and motion Collection, stagger, impact, compound." — The four-word value cards with staggered entry animation. This is the signature element.

## Cards & Animation

- "I think we can get back some of the previous card animation elements, but give them our own treatment." — Wants card animations but with a distinct RR identity, not generic.
- "It has too much parallax and movement of the cards" — Card tilt was too aggressive when applied everywhere.
- "I like this as signature effect for a separate element" — 3D tilt should be reserved for specific signature elements only (project cards, material stack), not applied broadly.
- "When there's buttons it feels funky too — I like that, but toned down." — Button tilt is good but needs to be very subtle. Y-axis only, ~1.2 degrees.

## Buttons

- "Elements like meaning buttons are still not what I think about this." — The original buttons didn't feel right. Needed rethinking. (Resolved by switching to Declaration voice — uppercase, weighted, manufactured precision with inner highlights.)

## Materials & Frosted Surfaces

- "The frosted surface is the hardest to get right here." — Frost treatments are the most difficult material to nail.
- "But the Artemis background looks great — just lovely." — The case study section with coarse grain background, big "ARTEMIS" type, and frosted overlay panel is approved.

## Specific Dislikes

- "I don't like the animated line in the Foundation segment on cards." — Remove the animated underline (`::after`) on value cards. No animated decorative lines on card hover.

## Animation & Motion

- "As for animations, I think we can add more character to the page overall." — Wants more animation personality, but targeted (not blanket parallax).

## Page Structure & System

- "Lets break all this down to components — I want this to be a system file I want to give to AI so it will build websites that would look like this." — The file should be structured as an AI-consumable reference. Components clearly labeled. Tokens documented. Usage rules in comments.
- "Create a test page with real world examples of applying our design." — Wants to see materials applied functionally in real contexts (landing pages, dashboards, articles, pricing, settings panels).
- "I want some more graphic elements for the Landing page." — Hero section needs visual interest beyond text — geometric SVGs, reticle marks.

## General Design Instincts (from project docs)

- Manufactured, not natural. LEGO, not wood.
- Semi-translucent matte surfaces.
- Machined metal quality.
- Playful reactivity, physical knobs/dials.
- Deliberateness and care as core values.
- Rejects: clutter, "LLM-generated" look, decorative non-function.
